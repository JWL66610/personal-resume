# 个人简历主页

> 🎨 利用 AI 编程助手（Codex CLI）创建的网页版个人简历，零手写代码，全程自然语言驱动开发。

## ✨ 特点

- **AI 协作开发**：通过自然语言描述需求与设计偏好，由 AI 生成完整 HTML/CSS/JS 代码，多轮对话迭代优化
- **7 个内容模块**：基本信息、教育背景、实习经历、项目经历、校园实践、获奖荣誉、自我评价
- **丰富交互动效**：粒子动画背景、由远及近渐变进场、时间轴布局、卡片 3D 悬停、标签飞入等
- **现代设计**：白底 + 克莱因蓝配色、华文中宋字体、渐变光晕艺术字
- **全端响应式**：桌面端、平板端、手机端自适应
- **零依赖部署**：单文件 HTML，双击即可在浏览器打开

## 🛠 技术栈

- HTML5 / CSS3 / JavaScript
- particles.js（粒子背景）
- AOS.js（滚动动画）
- GitHub Pages 部署

## 🚀 快速开始

```bash
# 克隆仓库
git clone https://github.com/JWL66610/personal-resume.git

# 直接用浏览器打开
start index.html
```

或直接访问在线版本（GitHub Pages 部署后生效）。

## 📝 AI 开发流程

1. **需求描述**：以自然语言告诉 AI 想要的布局、配色、动效风格
2. **代码生成**：AI 生成完整单文件 HTML
3. **反馈迭代**：通过对话调整细节（时间轴方向、动画速度、字体选择等）
4. **验证发布**：浏览器预览确认效果，推送到 GitHub
